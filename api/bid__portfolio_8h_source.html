25 /// @brief A BidPortfolio is a collection of bids as responses to requests for
26 /// resources and associated constraints on those bids.
27 ///
28 /// A BidPortfolio contains all the information corresponding to a bidder's
29 /// response to resource requests. It is a light wrapper around the set of bids
30 /// and constraints for a given bidder, guaranteeing a single bidder per
31 /// portfolio. Responses are grouped by the bidder. Constraints are assumed to
32 /// act over the entire set of possible bids.
