enum cyclus::toolkit::TimeSeriesType : int

Time series types to be used in the RecordTimeSeries() functions.

These types have the following units which must be adhered to strictly:

  • POWER [MWe]
  • ENRICH_SWU [kg SWU]
  • ENRICH_FEED [kg]
