enum cyclus::toolkit::TimeSeriesType

Time series types to be used in the RecordTimeSeries() functions.

These types have the following unit which must be adhered to strictly:

  • POWER [MWe]
  • ENRICH_SWU [kg SWU]
  • ENRICH_FEED [kg]
